Configuring SSH Key

  • Generate new SSH key pair.
  • Copy the SSH private key from the server and paste it to the "private_key.ppk" file on your machine.
  • Open "PuTTY gen" on your machine and load the "private_key.ppk" file and click "Generate" button.
  • Putty will display the "Public Key" upload to the server.
  • On PuTTY gen click save "Private Key" and enter a name while downloading.
  • Once downloaded right click the downloaded file and choose "Load in Pageant".
  • Start Putty, enter hostname,port, portcol.. once accessed enter username and pwd.
